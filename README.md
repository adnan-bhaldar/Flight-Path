### Run Locally

To run **Flight Path** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/adnan-bhaldar/Flight-Path.git
```

Windows:

```bash
git clone https://github.com/adnan-bhaldar/Flight-Path.git
```
